Google Sheets 연동 설정

본인의 구글 스프레드시트에 점수를 실시간 저장하려면 발급받은 Google Apps Script 웹앱 URL을 아래에 등록하세요.

💡 주의: URL을 입력하지 않으면 게임이 안전한 로컬 저장공간에 모의 점수 데이터를 저장합니다.

구글 시트 연동 가이드

STEP 1 구글 스프레드시트 만들기

1. Google Sheets로 이동하여 새로운 시트를 만듭니다.
2. 시트의 첫 번째 행에 다음과 같이 열 머리글을 지정해 주세요:

A열 B열 C열
Name Score Gems

STEP 2 Apps Script 코드 붙여넣기

1. 구글 스프레드시트 메뉴에서 확장 프로그램 > Apps Script를 클릭합니다.
2. 기존 코드를 모두 지우고 아래의 코드를 복사해서 붙여넣으세요. 

function doGet() {
  var sheet = SpreadsheetApp.getActiveSpreadsheet().getActiveSheet();
  var data = sheet.getDataRange().getValues();
  var result = [];
  
  // 첫 번째 줄(헤더) 제외하고 데이터 수집
  for (var i = 1; i < data.length; i++) {
    result.push({
      name: data[i][0],
      score: parseInt(data[i][1]) || 0,
      gems: parseInt(data[i][2]) || 0
    });
  }
  
  // 점수 기준 내림차순 정렬
  result.sort(function(a, b) { return b.score - a.score; });
  
  // 상위 10개만 리턴
  var top10 = result.slice(0, 10);
  
  return ContentService.createTextOutput(JSON.stringify(top10))
    .setMimeType(ContentService.MimeType.JSON);
}

function doPost(e) {
  var sheet = SpreadsheetApp.getActiveSpreadsheet().getActiveSheet();
  var params = JSON.parse(e.postData.contents);
  
  var name = params.name || "Unknown";
  var score = params.score || 0;
  var gems = params.gems || 0;
  
  sheet.appendRow([name, score, gems]);
  
  return ContentService.createTextOutput(JSON.stringify({ status: "success", name: name, score: score, gems: gems }))
    .setMimeType(ContentService.MimeType.JSON);
}

STEP 3 배포 및 URL 획득

1. Apps Script 편집기 우측 상단의 배포 > 새 배포를 클릭합니다.
2. 유형 선택에서 웹 앱(Web App)을 선택합니다.
3. 구성 설정을 다음과 같이 수행합니다:
  • 설명: 점수 데이터베이스 API
  • 다음 사용자 권한으로 실행: 나 (본인 이메일)
  • 액세스할 수 있는 사용자: 모든 사용자(Anyone)
4. 배포 버튼을 클릭하고 구글 보안 액세스 승인을 완료합니다.
5. 배포 완료 화면에서 제공되는 웹 앱 URL을 복사합니다.
6. 다시 이 게임 창으로 돌아와 우측 상단의 '시트 연동' 버튼을 눌러 복사한 URL을 입력해 주세요!
